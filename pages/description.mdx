# 專案說明

本專案的主要目標是為鋁門窗生產業者最佳化生產流程，旨在縮短其產品生產的總耗時（makespan），並且使產能平穩、精準滿足交貨期限、減少模具切換的時間以及盡量減少在製品庫存量。

其中，能夠精準滿足交貨時段尤為重要。以往的經驗中，缺乏精確的交貨時段估計導致了兩個主要問題：一是在未使用我們的估計模型之前，業務團隊可能因無法提供準確的交貨時段而失去潛在客戶；二是若給予的截止時段過晚，可能導致我們的產品在市場上的吸引力下降，客戶轉而尋找其他供應商。這些情況不僅影響了我們的業績，也妨礙了長期客戶關係的建立。

為解決這些問題並滿足目標，我們將首先收集現有的訂單數據，並將訂單轉換成製造需求單，這包括了製造需求單的數量、種類、各種產品的需求量以及可能存在的交貨期限。接著，我們將運用作業研究模型和自行開發的演算法來進行生產規劃以及估計新製造需求單完成的所需時間。此模型將考慮到各種因素，包括但不限於製造需求單種類（建設案、經銷案）、在製品種類、數量和工作站的的產能，以確保每個製造需求單都能在其截止時段前有效且準時地完成。

最後，我們會依據作業研究模型提供的生產規劃結果，設立一個模型，往回推算各前置作業的時程規劃（例如：叫貨的數量與時段）。

我們的模型主要聚焦於生產過程的生產規劃，透過精確的模型估計，我們旨在提供一個策略框架，幫助業者合理安排生產規劃，並在不讓舊製造需求單超過截止時段的前提下最小化新製造需求單的截止時段。此外，我們的模型將特別關注於避免生產過程中的任何製造需求單延期或提早情況，從而確保業者能夠滿足客戶需求、減少成本並維持市場競爭力。
